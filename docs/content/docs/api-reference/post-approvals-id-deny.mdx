---
title: POST /approvals/:id/deny
description: Deny and close a pending payment request
---

## Endpoint

```text
POST https://api.usezenithpay.xyz/approvals/:id/deny
```

## Auth

Bearer API key required.

## Request body

```json
{
  "humanSignature": "0x...",
  "timestamp": 1711234567890
}
```

Signature payload:

```json
{
  "action": "deny",
  "approvalId": "apr_01abc",
  "timestamp": 1711234567890
}
```

## Response shape

```json
{
  "id": "apr_01abc",
  "status": "denied",
  "merchant": "service.xyz",
  "amount": "0.50",
  "deniedAt": "2026-03-24T12:00:00.000Z"
}
```
