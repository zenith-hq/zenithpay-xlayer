---
title: GET /wallet/agents
description: List linked agent wallets for an owner, or inspect a specific agent
---

## Endpoint

```text
GET https://api.usezenithpay.xyz/wallet/agents
```

## Auth

Bearer API key required.

## Request

- Optional query: `address=0x...`
- Optional header: `X-Owner-Address: 0x...` (used to resolve owner-linked agents)

## Response shape

```json
{
  "agents": [
    {
      "address": "0x...",
      "label": "research-agent-01"
    }
  ]
}
```
