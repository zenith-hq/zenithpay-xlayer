---
title: GET /health
description: Service health and version status
---

## Endpoint

```text
GET https://api.usezenithpay.xyz/health
```

## Auth

None.

## Response shape

```json
{
  "status": "ok",
  "service": "zenithpay-api",
  "version": "1.0.0",
  "timestamp": "2026-03-24T12:00:00.000Z"
}
```
