ZenithPay's REST endpoints are described in `openapi.yaml` and grouped below for fast navigation.

## OpenAPI

- `openapi.yaml`: `/docs/api-reference/openapi.yaml`

## Endpoint groups

- **Health**: `GET /health`
- **Wallet**: `POST /wallet/genesis`, `GET /wallet/agents`, `GET /wallet/balance`
- **Payments**: `POST /pay`
- **Policy**: `GET /limits`, `POST /limits`
- **Approvals**: `GET /approvals`, `POST /approvals/:id/approve`, `POST /approvals/:id/deny`
- **Ledger**: `GET /ledger`
- **Seller x402**: `GET|POST /sell/agent-intel`
